# .NET Notes

## Overview

- .NET is a cross-platform developer platform for building many different types of applications
- It is open source but maintained by Microsoft
- Main languages: C#, F#, VB
- C++ is also supported for .NET by Microsoft
- Community languages: e.g., IronPython, Nemerle IronRuby, Boo, Oxygene
- These all compile to Intermediate Language (IL) that runs on the Common Language Runtime (CLR)

## Terms

- **MSI (Microsoft Installer)**: powerful legacy Windows app installer type that can be messy to uninstall
- **MSIX**: modern Windows app package format with virtualized registry access that needs to be signed for distribution
