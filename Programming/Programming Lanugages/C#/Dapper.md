# Dapper

1. **ORM (Object-Relational Mapper)**: a library that lets you work with a SQL database using objects instead of SQL
2. **Dynamic Type Parameter**: Dapper feature that allows you to pass parameters to SQL queries dynamically without defining a specific type
3. **Anonymous Type Parameter**: *Dapper feature that allows you to pass parameters to SQL queries using an anonymous type, providing a convenient way to define parameters inline without creating a separate class*
