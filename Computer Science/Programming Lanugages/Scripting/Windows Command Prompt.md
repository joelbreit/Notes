# Windows Command Prompt

## Common Commands

| Bash Command | Windows Command Prompt | PowerShell   |
| ------------ | ---------------------- | ------------ |
| `ls`         | `dir`                  | `ls` / `dir` |
| `cd`         | `cd`                   | `cd`         |
| `pwd`        | `cd`                   | `pwd`        |
| `touch`      | `echo. >`              | `New-Item`   |
| `cat`        | `type`                 | `cat`        |
| `echo`       | `echo`                 | `echo`       |
| `mkdir`      | `mkdir`                | `mkdir`      |
| `rmdir`      | `rmdir`                | `rmdir`      |
| `rm`         | `del`                  | `rm`         |
| `cp`         | `copy`                 | `cp`         |
